# StayAwake

Mantém o Windows acordado enquanto você usa um app específico — sem mexer nas configurações de energia, sem nada rodando em background.

Você configura uma vez. Depois disso, abre seus apps normalmente e o StayAwake faz o resto.

---

## Como usar

1. Abra o `StayAwake.exe`
2. Adicione os apps que você quer monitorar
3. Pronto — pode fechar. Seus apps continuam funcionando normalmente

Da próxima vez que abrir um dos apps configurados, o Windows vai parar de dormir automaticamente. Quando você fechar o app, tudo volta ao normal.

---

## Requisitos

- Windows 10 ou 11
- Permissão de administrador na primeira vez que abrir
